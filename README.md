## Ideia do projeto: 
O `Gerador de dados` é um projeto construído para as situações onde você precise gerar dados aleatórios que sejam válidos, onde você poderá gerar CPF's ou CNPJ's 100% válidos e senhas seguras, para que você use em seus projetos, por exemplo, ou apenas para testar funcionalidades.

Este é um projeto de estudos de front-end onde eu me desafiei para desenvolver uma aplicação com diversas rotas e colocá-la na internet para que outros pudessem acessar.

## Funcionalidades
Neste projeto, você possui as seguintes funcionalidades:
* Geração de CPF válido aleatório;
* Geração de CNPJ válido aleatório;
* Geração de senha, considerando tamanho e caracteres utilizados na senha;

## Tecnologias usadas:
Neste projeto foram usadas as seguintes tecnologias:
* `React:` Para criação do projeto em geral, utilizando componentes e useStates para controlar os dados exibidos da melhor maneira possível;
* `React-router:` Utilizado para criação das rotas do projeto, separando cada tipo de dado em sua específica rota;
* `Tailwindcss:` Para estilização do projeto sem necessidade de construção de arquivos CSS em particular;
* `Vercel:` Utilizei a plataforma vercel para exibir meu projeto na internet;

## Como usar o projeto:

Existem duas maneiras de utilizar, através da internet e localmente:

> ### Usando na internet 
Abra o navegador e digite a seguinte URL: https://gerador-de-dados-amber.vercel.app/

> ### Usando localmente

* Copie este repositório em seu computador;
* Execute o `npm install`
* Execute `npm run dev`
* Abra o navegador na porta indicada no terminal;
